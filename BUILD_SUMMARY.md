## 🎨 Color Customization

Modify the forest green in `css/styles.css`:

```css
:root {
  --accent: #2d5016;           /* Main forest green */
  --accent-light: #3d6b1f;     /* Lighter shade */
  --accent-hover: #1f3610;     /* Darker shade */
}
```

In dark mode (automatically generates):
```css
html.dark-mode {
  --accent: #4ade80;           /* Bright green for dark background */
  --accent-light: #86efac;     /* Light green */
  --accent-hover: #22c55e;     /* Medium green */
}
```
